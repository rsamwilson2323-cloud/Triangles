# 🔺 Neon Triangle Waves

An interactive HTML5 Canvas animation that creates glowing neon triangle waves with vibrant rainbow colors, smooth expansion effects, and dynamic light trails. Designed for both desktop and mobile devices, users can generate mesmerizing triangular ripple patterns through mouse and touch interactions.

## ✨ Features

* 🔺 Expanding neon triangle waves
* 🌈 Dynamic rainbow color transitions
* ✨ Glowing neon light effects
* 🖱️ Mouse click and drag support
* 📱 Mobile touch interaction
* 🎨 Smooth fading trails
* ⚡ Real-time Canvas rendering
* 🖥️ Fullscreen responsive design
* 🚀 Lightweight and dependency-free

## 🎮 Controls

### Desktop

* Click to create glowing triangle waves.
* Hold and drag to continuously generate expanding triangle effects.

### Mobile

* Tap anywhere to generate neon triangle ripples.
* Touch and drag to create continuous glowing wave animations.
